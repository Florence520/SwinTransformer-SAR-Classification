# SL-MULNet 前端界面

基于 PyQt5 实现的 SL-MULNet 项目前端界面，提供训练和推理两个主要模块。

## 功能模块

### 1. 模型训练模块
- 数据集路径选择
- 配置文件选择
- 预训练模型选择（可选）
- 训练参数设置（批次大小、学习率、训练轮数）
- 训练进度显示
- 实时日志输出

### 2. 推理预测模块
- 测试数据路径选择
- 模型文件选择
- 推理结果输出

### 3. 结果可视化模块
- 图像选择
- 原始图像显示
- 分割结果显示
- 结果保存

## 界面说明

### 侧边栏导航
- **训练按钮**：切换到模型训练模块
- **推理按钮**：切换到推理预测模块
- **可视化按钮**：切换到结果可视化模块
- **设置按钮**：打开设置界面

### 训练模块操作步骤
1. 点击「选择数据集」按钮，选择数据集目录
2. 点击「选择配置」按钮，选择训练配置文件（.py）
3. 可选：点击「选择模型」按钮，选择预训练模型（.pth）
4. 设置训练参数（批次大小、学习率、训练轮数）
5. 点击「开始训练」按钮启动训练
6. 点击「停止训练」按钮可中途停止

### 推理模块操作步骤
1. 点击「选择数据」按钮，选择测试数据目录
2. 点击「选择模型」按钮，选择训练好的模型文件
3. 点击「开始推理」按钮启动推理

### 可视化模块操作步骤
1. 点击「选择图像」按钮，选择要可视化的图像
2. 选择模型（如需）
3. 点击「开始生成」按钮生成分割结果
4. 点击「保存图片」按钮保存结果

## 注意事项

1. 确保已正确安装 PyQt5 依赖
2. 训练和推理需要在项目根目录下运行，确保路径正确
3. 首次运行可能需要等待模型加载
4. 训练过程中请勿关闭窗口，否则训练会中断
